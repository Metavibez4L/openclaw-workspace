# Self-Edit Skill

Use `/self-edit append <text>` to append text to this skill's SKILL.md.
Use `/self-edit replace <old> <new>` to replace text in SKILL.md.

## Examples
- `/self-edit append This skill can now update itself.`
- `/self-edit replace oldtext newtext`
