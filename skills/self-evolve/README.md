# Self-Evolve Skill (Demo)

This directory contains a minimal self-evolving automation script.

## Usage

- `./self_evolve.sh append <text>` — Append text to this script.
- `./self_evolve.sh replace <old> <new>` — Replace text in this script.
- `./self_evolve.sh doc-append <text>` — Append text to README.md.
- `./self_evolve.sh doc-replace <old> <new>` — Replace text in README.md.

> This is a demonstration of a self-modifying script. Use with care!
